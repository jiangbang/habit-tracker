# 计划坚持完成记录 🎯

记录每天坚持完成的目标和习惯，追踪成长轨迹。

## 📋 目标

- 记录每日/每周需要坚持的习惯
- 追踪完成情况和连续天数
- 反思和改进

## 📁 目录结构

```
habit-tracker/
├── README.md           # 项目说明
├── habits/             # 习惯记录
│   └── 2026-03.md      # 按月记录
├── goals/              # 长期目标
│   └── 2026-goals.md   # 年度目标
└── reviews/            # 定期复盘
    └── weekly/         # 周复盘
```

## 🚀 使用方法

### 记录每日习惯

在 `habits/YYYY-MM.md` 文件中添加记录：

```markdown
## 2026-03-16

- [x] 早起 (6:00)
- [x] 运动 30 分钟
- [ ] 阅读 30 分钟
- [x] 写代码

今日总结：今天状态不错，完成了大部分目标。
```

### 设置目标

在 `goals/2026-goals.md` 中写下你的年度目标。

### 每周复盘

在 `reviews/weekly/` 中写周复盘，总结收获和改进点。
